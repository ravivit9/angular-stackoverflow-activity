Angular StackOverflow Activity

AngularJS module for showing off your public StackOverfow activity.

Getting Started.

1. Use the StackoverflowActivityService method provided to fetch your data.

StackoverflowActivityService.events({
  user:'1113921',
  params:{
    key:'n0t4r34l4cc3sst0k3n123',
    site:'stackoverflow',
    filter:'unsafe',
    callback: 'JSON_CALLBACK'
  }
}).get().$promise.then(function(events){
  $scope.activity = events.data;
});

$scope.config = {
  limit: 4;
}

2. Scope it in the callback and use it with the stackoverflow-activity directive.

<div stackoverflow-activity events="activity" options="config"></div>
